# 📊 Corporate Data Analyzer (Smart Analyzer)

Corporate Data Analyzer is a desktop-based data analysis application built using Python that enables users to quickly analyze, visualize, and export data from CSV and Excel files without requiring any programming knowledge.

The application provides an intuitive graphical interface where users can upload datasets, generate summarized reports using Group By and Aggregation functions, and visualize insights through multiple chart types. It is designed to simplify data analysis tasks for business users, students, and professionals.

## 🚀 Key Features

* Import data from CSV and Excel files
* Automatic detection of text and numeric columns
* Generate reports using Group By and Aggregation functions (Sum, Mean, Max, Min, Count, Median)
* Interactive data visualization with charts:

  * Bar Chart
  * Column Chart
  * Line Chart
  * Pie Chart
* Real-time report preview in tabular format
* Export reports to Excel (.xlsx) or CSV (.csv)
* Export charts as PNG images
* Clean and user-friendly graphical interface

## 🖥️ Technologies Used

* Python
* Tkinter (for GUI)
* Pandas (for data processing)
* Matplotlib (for data visualization)

## ⚙️ How It Works

1. Select a CSV or Excel file using the Browse option
2. Click "Read" to load and analyze the dataset
3. Choose:

   * Group By column
   * Aggregation type
   * Value (numeric column)
4. Click "Preview Report" to generate summary
5. Use "Preview Chart" to visualize the data
6. Export report or chart as needed

## 📦 Installation

Install required libraries:

pip install pandas matplotlib openpyxl

## ▶️ Run the Application

python smart_analyzer.py

## 🛠️ Convert to Executable

pyinstaller --onefile --windowed smart_analyzer.py

## 📁 Output

* Report files are saved as Excel or CSV
* Charts are saved as PNG
* Files are stored in the same directory as the input file
